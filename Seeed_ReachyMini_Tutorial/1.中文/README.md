# ReachyMini 桌面机器人课程：从基础操作到二次开发

## 课程简介
欢迎参加 **ReachyMini 桌面机器人课程**！本课程专为机器人开发初学者和教育工作者设计，旨在通过系统的理论讲解与实战演练，帮助您快速掌握 ReachyMini 桌面机器人的核心控制技术和二次开发能力。

ReachyMini 是一款功能强大的教育机械臂，配备头部、天线、身体等多个可控部件，支持摄像头、音频、IMU 等丰富的传感器接口。本课程将带您从基础操作开始，逐步深入到运动学、插值、异步控制等高级技术，为构建复杂的机器人应用打下坚实基础。

### 您将学到：
* 如何连接和配置 ReachyMini 机器人（包括网络连接和仿真模式）。
* 掌握机器人头部、天线、身体各部件的精确控制方法。
* 学习摄像头和音频等多媒体功能的开发与应用。
* 深入理解机器人的运动学（正逆运动学）和坐标变换。
* 掌握轨迹插值和动作记录回放技术。
* 学习异步控制和并行任务处理。
* 通过丰富的实战案例提升开发能力。

## 如何开始
1. 确保您已准备好 ReachyMini 机器人硬件或仿真环境。
2. 从 [1.快速开始](./1.快速开始/README.md) 开始您的学习之旅。
3. 建议按照章节顺序学习，逐步掌握各项功能。
4. 每个章节都包含详细的代码示例和实践练习。

## 环境要求

* **操作系统**: Ubuntu 20.04 / 22.04 或 Windows 10/11
* **Python 版本**: Python 3.8 或更高版本
* **硬件**: ReachyMini 桌面机器人（可选，支持仿真模式）
